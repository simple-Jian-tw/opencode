## Icons (SVG)

### Heroicons
- **GitHub**: https://github.com/tailwindlabs/heroicons
- **Raw URL Pattern**: `https://raw.githubusercontent.com/tailwindlabs/heroicons/master/src/24/{style}/{name}.svg`
- **Styles**: `outline`, `solid`, `mini`
- **Usage**: Tailwind official icons, optimized SVG, React/Vue components available

### Remix Icon
- **GitHub**: https://github.com/Remix-Design/RemixIcon
- **Raw URL Pattern**: `https://raw.githubusercontent.com/Remix-Design/RemixIcon/master/icons/{category}/{name}.svg`
- **Categories**: Buildings, Business, Communication, Design, Development, Device, Document, Editor, Finance, Health, Logos, Map, Media, Others, System, User, Weather
- **Count**: 2000+ neutral style icons

### Tabler Icons
- **GitHub**: https://github.com/tabler/tabler-icons
- **Raw URL Pattern**: `https://raw.githubusercontent.com/tabler/tabler-icons/main/icons/{style}/{name}.svg`
- **Styles**: `outline`, `filled`
- **Count**: 5000+ pixel-perfect icons

## Media Resources (API)

### Pexels
- **API Docs**: https://www.pexels.com/api/documentation/
- **Features**: Photos, Videos, search by keyword/color/orientation
- **Auth**: Free API key required
- **Response**: JSON with direct download URLs (.jpg, .mp4)

### Pixabay
- **API Docs**: https://pixabay.com/api/docs/
- **Features**: Photos, Illustrations, Vectors, Videos
- **Auth**: Free API key required
- **Response**: JSON with multiple resolution URLs

### Freesound
- **API Docs**: https://freesound.org/docs/api/
- **Features**: Sound effects, ambient sounds, UI sounds
- **Search Examples**: "click", "ui", "notification", "ambient"
- **Auth**: OAuth2 required for downloads

### Tosound (淘声网)
- **Website**: https://www.tosound.com/
- **Search URL**: `https://www.tosound.com/search/{keyword}`
- **License**: CC0 (public domain)
- **Language**: Chinese interface

## Illustrations & Animations

### Illlustrations
- **GitHub**: https://github.com/realvjy/illlustrations
- **Formats**: AI, SVG, PNG
- **License**: Open source, free for commercial use

### Lottie Web
- **GitHub**: https://github.com/airbnb/lottie-web
- **Format**: JSON animations
- **Usage**: Parse JSON, render with lottie-web player
- **Resource**: LottieFiles public library for pre-made animations

### unDraw
- **Website**: https://undraw.co/illustrations
- **Format**: SVG with customizable colors
- **Usage**: Change fill color via URL parameter or DOM manipulation

## Developer Tools

### Squoosh (Image Compression)
- **Website**: https://squoosh.app/
- **CLI**: `npx @squoosh/cli`
- **Usage**: Batch compress images via command line
- **Formats**: WebP, AVIF, JPEG, PNG optimization

### Public APIs List

- **Categories**: Animals, Anime, Anti-Malware, Art, Books, Business, Calendar, Cloud, Cryptocurrency, Currency, Data, Development, Entertainment, Environment, Events, Finance, Food, Games, Geocoding, Government, Health, Jobs, Machine Learning, Music, News, Open Data, Patent, Personality, Phone, Photography, Science, Security, Shopping, Social, Sports, Test, Text, Tracking, Transportation, URL, Video, Weather

### gitignore.io
- **Website**: https://gitignore.io/
- **API**: `https://www.toptal.com/developers/gitignore/api/{techs}`

## CSS & Animation Libraries

### React Bits
- **Docs**: https://reactbits.dev/docs/introduction
- **Features**: Glitch text, Circular text, advanced animations
- **Usage**: Copy component code, customize parameters

### CSS Loaders
- **Website**: https://css-loaders.com/
- **Count**: 600+ loading animations
- **Format**: Pure HTML/CSS
- **Usage**: Inspect page, extract CSS for desired loader

### Anime.js
- **GitHub**: https://github.com/juliangarnier/anime
- **Docs**: README contains full API reference
- **Features**: Timeline animations, staggering, SVG morphing
- **Syntax**: `anime({ targets: '.element', translateX: 250, duration: 800 })`

### Spider Cloud (Web Scraping API)
- **API Docs**: https://spider.cloud/docs/api
- **Features**: Web scraping, data extraction, Markdown conversion
- **Usage**: Fetch and clean web content programmatically
